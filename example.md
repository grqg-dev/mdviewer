# Hello, mdviewer

A **glow-inspired** markdown viewer for the terminal.

## Features

- Renders markdown with style
- Scroll with `j`/`k`, arrows, `space`, `b`, `f`
- Press **Esc** to quit

    println!("Hello, world!");
